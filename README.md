# LeafletServer
Server for Zarkoix/Leaflet

## Requirements
* python3
* (all of the Leaflet dependacies)

## Installation:
* pip3 install django
* pip3 install django_webpack_loader
* (note: use of a "virtualenv" is reccomended)

## Editing:
* use tabs, not spaces
* pylint --load-plugins pylint_django

## Run server:
python3 manage.py runserver
